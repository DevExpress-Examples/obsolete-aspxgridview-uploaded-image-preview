OBSOLETE - Inserting a new row in ASPxGridView with the image preview enabled

UPDATED: Starting with version v2015 vol 1 (v15.1), this functionality is available out of the box. Simply set the GridViewDataBinaryImageColumn.PropertiesBinaryImage.EditingSettings.Enabled property to True to activate it. Please refer to the ASP.NET Data Grid - Binary Image Editor (Coming soon in v15.1) (https://community.devexpress.com/blogs/aspnet/archive/2015/05/28/asp-net-data-grid-binary-image-editor-coming-soon-in-v15-1.aspx) blog post and the Binary Image Column Editing (http://demos.devexpress.com/ASPxGridViewDemos/GridEditing/BinaryImageColumnEditing.aspx) demo for more information. If you have version v15.1+ available, consider using the built-in functionality instead of the approach detailed below. This example shows how to use ASPxUploadControl on the ASPxGridView's Edit Form to preview an image before performing data update.
